## Installation

Install node_modules with npm

```bash
  npm install
```
    
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/moerabaya/magazine-landing-page
```

Go to the project directory

```bash
  cd magazine-landing-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
